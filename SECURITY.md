# Security

These skills are shared outside ClawHub, so treat them as source code you should review before installing.

Before installing a skill:

- Read `SKILL.md`.
- Review scripts and any referenced helper files.
- Check required binaries and environment variables.
- Do not paste secrets into a chat or setup command unless you understand how they will be used.
- Prefer installing one skill at a time and testing it with a harmless request first.

If you find a sensitive path, token, credential, prompt injection, or unsafe behavior in this repository, open an issue or contact the maintainer before using the skill.
